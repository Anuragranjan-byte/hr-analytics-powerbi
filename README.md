# HR Analytics Dashboard (Power BI)

## Project Overview
This project analyzes HR data using Power BI to understand employee distribution, salary patterns, and workforce demographics.

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization
- Dashboard Design

## Key Insights
- Gender distribution across employees
- Average salary by job role
- Employee count per job role
- Age distribution of employees
- Education qualification breakdown
